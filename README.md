# TO-DO-LIST

https://to-do-list-three-henna.vercel.app/

A simple, responsive frontend application for managing tasks. It supports adding tasks with date and time selection via a calendar and clock interface.

## Features
- Add, edit, and delete tasks
- Calendar date picker (defaults to today)
- Time picker (clock interface or manual input)
- Smooth modal workflow for task creation
- Responsive design with Tailwind CSS + DaisyUI
- No backend or database â€” tasks are stored in memory/local storage

## Tech Stack
- **HTML**
- **CSS** (Tailwind CSS + DaisyUI)
- **JavaScript** (Vanilla ES6)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/koshal50/TO-DO-LIST.git
   ```
2. Open the project folder:
   ```bash
   cd TO-DO-LIST
   ```
3. Open `index.html` in your browser.  
   No backend setup is required.

## Usage
- Add a new task using the input field.
- Choose a date from the calendar (defaults to today).
- Select a time using the clock input (optional).
- Edit or delete tasks using the action buttons.
- Tasks persist in local storage (if enabled in the app).
